  • Anteriormente, isso era feito com cookies. No entanto, Web Storage é mais seguro e mais rápido. Os dados não estão incluídas com cada solicitação do servidor, mas usados ​​apenas quando solicitado. Também é possível armazenar grandes quantidades de dados, sem afetar o desempenho do site.

Uma Breve descrição sobre as novas tags

    <header> <nav> <section> <article> <aside> <footer>
  • O elemento header pode ser utilizado para agrupar índices de conteúdo, campos de busca, cabeçalho do site com titulo ou logo.

  • O elemento nav é utilizado para representar uma seção da página que contém links para outras partes do site.normalvmente utilizados em menus de navegação.

  • A tag section define uma nova seção do site, uma área genérica do site. Por exemplo a home pode ser dividida em destaque, novidades, chamada para conteúdo completo.

  • O elemento article é utilizado para representar o conteúdo do site propriamente dito, como posts, artigos e outros textos em geral.

  • A tag aside especifica o conteúdo relacionado com o artigo dentro de um artigo ou página web

  • A tag footer especifica um rodapé para a seção de uma página

Responsive Design

      O que design responivo?

    • Para alguns e um termo novo mas não se engane ele já esta em discurção a muito tempo

      Porque usar?

    • Nos tempos atuais, além dos desktops e notebooks, é possível acessar a internet através de smartphones, tablets, televisores

      Como fazer?

    • O Design Responsivo usa Media Queries.São expressões que direcionam o visitante do seu site para uma folha de estilos diferente de acordo com o dispositivo que ele estiver utilizando.
